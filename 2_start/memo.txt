ts는 바로 실행이 불가하므로 ts => js로 컴파일 필요
- 컴파일된 코드에서는 타입정의가 제거되어 있음

npm install typescript -g
- tsc 명령어를 사용하기 위해 글로벌 설치
- 번들러로 tsc를 돌려주면 체계적으로 전체프로젝트 컴파일 가능

tsc로 컴파일할 때 부가적인 옵션을 tsconfig를 통해 설정가능

"allowJs": true,
"checkJs": true,
"noImplicitAny": true
